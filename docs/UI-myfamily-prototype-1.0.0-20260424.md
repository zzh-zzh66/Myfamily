# MyFamily - UI原型设计

## 设计概述

**设计风格**: 中国风水墨画
**色彩主题**: 墨韵红梅（朱砂红 + 宣纸白 + 靛蓝）
**字体**: 思源宋体（标题）、思源黑体（正文）

---

## 原型目录

| 页面 | 文件 | 说明 |
|------|------|------|
| 登录注册 | UI-01-login | 登录/注册页 |
| 首页/族谱树 | UI-02-home | 族谱树展示 |
| 成员管理 | UI-03-member | 成员列表/详情 |
| 家族信箱 | UI-04-mail | 邮件收发 |
| 家族动态 | UI-05-post | 论坛/动态 |
| 纪念堂 | UI-06-memorial | 纪念堂展示 |
| 个人主页 | UI-07-profile | 成员主页 |

---

## 附录：组件状态规范

### 按钮状态

| 状态 | 样式 |
|------|------|
| 默认 | 朱砂红渐变背景 |
| 悬停 | 颜色加深 + 上浮阴影 |
| 点击 | 颜色更深 + 下沉 |
| 禁用 | 灰色背景，无交互 |

### 输入框状态

| 状态 | 样式 |
|------|------|
| 默认 | 淡墨边框 |
| 聚焦 | 朱砂红边框 + 外发光 |
| 错误 | 红色边框 + 错误提示 |
| 禁用 | 灰色背景 |

### 导航状态

| 状态 | 样式 |
|------|------|
| 默认 | 淡墨文字 |
| 悬停 | 背景淡红 + 文字加深 |
| 选中 | 左侧朱砂红竖线 + 朱砂红文字 |
